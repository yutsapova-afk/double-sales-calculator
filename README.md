# Double Sales Calculator

Калькулятор системы продаж для инфобизнеса.

## Возможности
- Расчёт конверсий воронки продаж
- Прогноз доходов на год (3 сценария)
- Учёт всех расходов
- Генерация PDF-отчёта
- Адаптивный дизайн

## Локальный запуск

```bash
npm install
npm start
```
